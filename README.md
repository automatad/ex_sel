# ExSel

Simple expression language for elixir.

## Installation

The package can be installed by adding `ex_sel` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:ex_sel, "~> 0.1.0"}
  ]
end
```

Documentation can be found at [https://hexdocs.pm/ex_sel](https://hexdocs.pm/ex_sel).
